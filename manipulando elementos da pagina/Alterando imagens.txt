Olá!   Nessa aula, aprendemos a alterar imagens no nosso projeto usando JavaScript. Primeiro, identificamos a tag <img> no HTML e a referenciamos no JavaScript usando document.querySelector(). Depois, usamos o método setAttribute() para modificar o atributo src da imagem, que define o caminho para a nova imagem. Fizemos isso para cada botão do temporizador, alterando a imagem de acordo com o tipo de descanso selecionado. No final, vimos que o código está um pouco repetitivo e que podemos refatorá-lo para torná-lo mais eficiente. Na próxima aula, vamos aprender a organizar melhor nosso código com funções e métodos! 

Imagine que você está montando um álbum de fotos e quer colocar fotos diferentes em cada página. Cada página representa uma tela do seu projeto, e as fotos são as imagens que você quer exibir.No JavaScript, você pode usar o document.querySelector() como se estivesse procurando a página específica do álbum que você quer editar. Depois, você usa o setAttribute() para "colar" a foto desejada naquela página.É como se você estivesse dizendo: "Essa página aqui, quero que a foto seja essa, e não aquela que estava lá antes". No nosso caso, cada botão do temporizador é como um botão que te leva para uma página diferente do álbum. Ao clicar em cada botão, você muda a foto da página, assim como no nosso projeto, ao clicar em cada botão do temporizador, a imagem muda de acordo com o tipo de descanso selecionado.Essa aula foi como aprender a "colar" fotos no álbum, e na próxima aula, vamos aprender a organizar melhor o álbum, criando pastas para cada tipo de foto, para que fique mais fácil encontrar e trocar as fotos!
