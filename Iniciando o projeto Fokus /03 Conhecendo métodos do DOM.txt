Olá!   Nessa aula, aprendemos como capturar elementos HTML na tela usando o console do navegador. Vimos como usar os métodos document.querySelector(), document.querySelectorAll(), document.getElementsByClassName() e document.getElementById() para selecionar elementos específicos. Também exploramos o conceito de DOM (Document Object Model), que é uma estrutura em forma de árvore que representa o conteúdo da página web. Com esses métodos, podemos manipular os elementos HTML, como botões, textos e imagens, para criar páginas web mais interativas e dinâmicas. Você se lembra de como usar cada um dos métodos que aprendemos?   Que tal praticar um pouco? Tente usar esses métodos para selecionar os botões "Descanso curto" e "Descanso longo" no projeto Fokus. 

Imagine que você está construindo uma casa de Lego. Cada tijolinho representa um elemento HTML, como um botão, uma imagem ou um parágrafo de texto.  O DOM seria como o manual de instruções da sua casa de Lego, que organiza todos os tijolinhos em uma estrutura hierárquica.Os métodos que aprendemos são como ferramentas que te permitem pegar os tijolinhos específicos que você precisa para construir sua casa. 
document.querySelector() seria como uma lupa que você usa para encontrar o primeiro tijolinho que você precisa. Você pode procurar por um tijolinho específico, como um botão com a cor vermelha, ou por um tijolinho com um número específico.
document.querySelectorAll() seria como uma caixa de ferramentas que te permite pegar todos os tijolinhos de um tipo específico, como todos os tijolinhos vermelhos ou todos os tijolinhos com um número específico.
document.getElementsByClassName() seria como um filtro que te permite pegar todos os tijolinhos que possuem uma etiqueta específica, como "janela" ou "porta".
document.getElementById() seria como um mapa que te permite encontrar um tijolinho específico com um nome único, como "porta da frente".

Com essas ferramentas, você pode pegar os tijolinhos que precisa para construir sua casa de Lego, e com o DOM, você pode organizar todos os tijolinhos em uma estrutura lógica para criar uma casa completa e funcional.No caso do projeto Fokus, você está usando JavaScript para "construir" uma aplicação web interativa. Os métodos que aprendemos te permitem pegar os elementos HTML que você precisa para criar essa aplicação, como os botões de "Descanso curto" e "Descanso longo".
