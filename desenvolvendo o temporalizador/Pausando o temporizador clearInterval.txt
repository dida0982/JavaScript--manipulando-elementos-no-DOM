Olá!   Nessa aula, aprendemos como pausar e zerar o temporizador do nosso projeto Fokus, usando o JavaScript. Primeiro, criamos uma função chamada zerar() que usa clearInterval() para interromper a contagem e define intervaloId como null. Depois, modificamos a função iniciar() para que ela também possa pausar o temporizador. Se intervaloId tiver um valor, a função zerar() é chamada para interromper a contagem. Por fim, renomeamos a função iniciar() para iniciarOuPausar(), pois ela agora é responsável por iniciar e pausar o temporizador. Agora, nosso temporizador pode ser iniciado, pausado e zerado! Quando você aperta o botão "Começar", o relógio começa a contar o tempo.  A função iniciarOuPausar() é como o botão do relógio:  você aperta uma vez para começar a contar, e aperta de novo para pausar.  A função zerar() é como o botão "Reset" do relógio:  ele para a contagem e volta o tempo para zero.O intervaloId é como a "chave" que liga o relógio.  Quando você aperta "Começar", a chave é ligada e o relógio começa a funcionar.  Quando você aperta "Pausar", a chave é desligada e o relógio para.  O clearInterval() é como um "interruptor" que desliga o relógio.  Quando você chama clearInterval(), o relógio para de funcionar.Entendeu?  É como se o JavaScript estivesse controlando o relógio, dizendo quando ele deve começar, parar e zerar. Espero que essa analogia tenha te ajudado a entender melhor!
