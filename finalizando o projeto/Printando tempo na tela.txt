Olá!   Nessa aula, aprendemos a exibir o temporizador na tela do nosso projeto! Primeiro, identificamos a div no HTML onde o temporizador deve aparecer, com o ID timer. Depois, criamos uma variável no JavaScript chamada tempoNaTela para capturar essa div usando document.querySelector('#timer').Criamos uma nova função chamada mostrarTempo() para exibir o tempo na tela. Essa função pega o valor do tempo armazenado na variável tempoDecorridoEmSegundos e o insere na div tempoNaTela usando o método innerHTML.Para que o tempo seja atualizado durante a contagem regressiva, chamamos a função mostrarTempo() dentro da função contagemRegressiva().Por fim, chamamos a função mostrarTempo() no escopo global para garantir que o tempo seja exibido na tela mesmo antes de iniciar a contagem regressiva.Agora, o temporizador está aparecendo na tela, mas ainda precisamos formatá-lo para exibir os minutos e segundos corretamente. 

Imagine que você está construindo um relógio de parede. Você já tem todas as engrenagens que fazem o relógio funcionar, mas ainda precisa de um mostrador para mostrar as horas. Nessa aula, estamos fazendo algo parecido com o nosso temporizador! Já temos a lógica que faz a contagem regressiva, mas precisamos de um lugar para exibir o tempo que está passando.A div com o ID timer no HTML é como o mostrador do relógio. É o espaço onde o tempo será exibido.A variável tempoNaTela no JavaScript é como um ponteiro que aponta para o mostrador. Ela nos permite acessar e modificar o conteúdo da div timer.A função mostrarTempo() é como o mecanismo que transforma o tempo em números visíveis no mostrador. Ela pega o tempo que está sendo contado e o coloca no mostrador para que possamos ver.Chamar a função mostrarTempo() dentro da contagemRegressiva() é como conectar o mecanismo do relógio ao mostrador. A cada segundo que passa, o mecanismo atualiza o mostrador para mostrar o tempo correto.Chamar a função mostrarTempo() no escopo global é como colocar o relógio na parede. Ele está pronto para mostrar o tempo, mesmo antes de começar a funcionar.Agora, nosso temporizador está pronto para mostrar o tempo, mas ainda precisamos formatá-lo para que ele apareça como "25:00" (25 minutos e 0 segundos), em vez de apenas "1500" (1500 segundos). É como se o mostrador do nosso relógio estivesse mostrando apenas os segundos, e precisamos ajustá-lo para mostrar horas e minutos!
